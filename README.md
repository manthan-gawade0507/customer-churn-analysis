# Customer Churn Prediction Project

## 📌 Project Overview
This project predicts customer churn using Machine Learning and provides business insights using Power BI.

## 🛠 Tools Used
- Python (Pandas, Scikit-learn)
- Power BI
- Machine Learning (Random Forest)

## 📊 Project Components
- ML Model → churn_model.ipynb
- Dashboard → churn_dashboard.pbix
- Presentation → churn_presentation.pptx

## 📈 Result
Random Forest achieved best performance and helps identify high-risk customers.

## 💼 Business Impact
Helps reduce customer churn and improve retention strategies.
